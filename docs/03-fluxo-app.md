# Fluxo do Aplicativo - Liga dos Bruxos

## 1. Tela Inicial

- Logo do aplicativo
- Botão de login
- Botão de cadastro

---

## 2. Login

- Usuário informa e-mail e senha
- Sistema valida credenciais
- Redireciona para o painel correspondente:
  - Administrador
  - Jogador

---

## 3. Fluxo do Jogador

### 3.1 Dashboard
- Próximo jogo
- Status de pagamento
- Ranking atual
- Última avaliação

### 3.2 Confirmar Presença
- Confirmar
- Não vai
- Entrar na lista de espera

### 3.3 Histórico
- Jogos anteriores
- Notas recebidas
- Estatísticas

---

## 4. Fluxo do Administrador

### 4.1 Dashboard Administrativo
- Lista de jogadores
- Próximo jogo
- Status financeiro
- Ranking geral

### 4.2 Gestão de Jogos
- Criar novo jogo
- Editar jogo
- Encerrar jogo

### 4.3 Formação de Times
- Gerar times automaticamente
- Ajustar manualmente

### 4.4 Controle Financeiro
- Confirmar pagamento
- Visualizar inadimplentes
- Registrar pagamento manual

### 4.5 Ranking
- Atualização automática
- Visualização anual

---

## 5. Encerramento de Jogo

- Inserção do placar
- Registro de gols
- Registro de assistências
- Abertura para avaliação
- Cálculo automático de notas
- Atualização do ranking
