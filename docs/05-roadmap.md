# Roadmap do Projeto - Liga dos Bruxos

## Visão Geral

Este roadmap define as etapas de desenvolvimento do aplicativo, considerando dedicação diária no turno da manhã.

---

## Fase 1 – Planejamento e Documentação (Semana 1)

- Definição de requisitos
- Definição de regras de negócio
- Estrutura de navegação
- Arquitetura do sistema
- Planejamento de funcionalidades futuras

Status: Em andamento

---

## Fase 2 – Estrutura Inicial do Projeto (Semana 2)

- Criação do projeto com Expo
- Organização de pastas
- Configuração inicial
- Primeira tela funcionando

Objetivo: Aplicativo abrindo corretamente no celular

---

## Fase 3 – Funcionalidades Base (Semanas 3 e 4)

- Cadastro de usuários
- Tela de login
- Cadastro de jogadores
- Criação de jogos
- Lista de presença

Objetivo: MVP funcional

---

## Fase 4 – Sistema de Pontuação (Semana 5)

- Registro de placar
- Registro de gols e assistências
- Encerramento automático
- Atualização de ranking

Objetivo: Diferencial competitivo implementado

---

## Fase 5 – Sistema Financeiro (Semana 6)

- Controle de pagamentos
- Confirmação manual pelo administrador
- Controle de inadimplência

---

## Fase 6 – Avaliação de Jogadores (Semana 7)

- Sistema de votação
- Cálculo de média excluindo maior e menor nota
- Atualização automática da média

---

## Fase 7 – Ajustes e Melhorias (Semana 8)

- Testes completos
- Correção de bugs
- Melhorias de interface
- Preparação para apresentação

---

## Fase 8 – Preparação para TCC

- Documentação final
- Diagramas UML
- Relatório técnico
- Apresentação
